HTML Forms

We need to add a name attribute to form elements so that when the form is submitted the data can be matched to the name. for multiple options where the name is the same you must add value attribute.

When we submit the form in our test example the data is sent to the url as a query string using the GET method.We would normally use POST method and data would be sent securely.

The query string with mistakes

?firstName=dom&lastName=ztm&email=test%40ztm.com&password=password&birthday=1975-01-01&gender=on&cat=on

The query string with fixes

firstName=Dom&lastName=ztm&email=test%40ztm.com&password=pass1234&birthday=1975-01-01&gender=male&dog=on&cars=audi
